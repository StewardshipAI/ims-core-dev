# ECSC Calculation Specification

ECSC = (Estimated Tokens × Price) / P(success)

Token Estimation:
- Input tokens
- Expected output tokens
- Safety margin (default 20%)

P(success):
- Capability fit score
- Historical adjustment (optional)

Lower ECSC wins.
